# 英语学习方式和工具
 - 词缀词根 ： 我个人比较认可的方式，一个词根关联记忆多个单词。但不能用于每个单词。
 - 语音变种： 有一些小用，我们汉语不也有音相同导致新用词的情况吗。
 - 娱乐文章： 单词以一种奇怪姿势进入脑袋。
 - 翻译笔： 为避免读文章时用手机或者字典翻译单词，总是打断注意力。 就不说品牌了，有像我一样阅读文章时总拿起手机字典情况的，可以买一个试试。
